# Android-AnimationBanner

##控件简介 
  AnimationBanner是一个利用属性动画原理实现的横幅广告控件。
  
##AnimationBanner特点
  1.实现原理及使用都很简单<br/>
  2.可灵活设置Banner数量，1张或N张<br/>
  3.附带Indicator小圆点指示器，并自动联动<br/>
  4.可指定Indicator指示器在Banner中的位置<br/>
  5.可配置动画持续时间来调整Banner滑动的速度<br/>
  6.可自定义Banner的大小以适配图片<br/>
  7.可响应Banner滑动与点击事件
